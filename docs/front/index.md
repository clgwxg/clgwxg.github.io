# 前端

前端主要使用的技术有vue3、pinia、vue-router、naive-ui

## 环境安装

### node 安装

node 版本 20.9.0 [链接](https://nodejs.org/en)

### pnpm安装

node 安装后，使用npm命令安装pnpm

```md
npm install -g pnpm
```

## 项目下载执行

### clone

```md
git clone https://github.com/clgwxg/gf-naive-template.git
```

### 安装依赖

```md
pnpm install
```

### 开发环境

```md
pnpm dev
```
